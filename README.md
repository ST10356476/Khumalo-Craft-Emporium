# Khumalo Craft Emporium

**Khumalo Craft Emporium** is a web-based application designed to showcase and sell hand-crafted items. The platform offers a user-friendly interface for browsing, purchasing, and managing artisan crafts.

---

## Features
- **Product Listings**: View a catalog of handmade crafts with descriptions, prices, and images.
- **Shopping Cart**: Add items to a cart and manage purchases easily.
- **Responsive Design**: Works seamlessly across desktop and mobile devices.
- **Admin Panel**: Manage products, orders, and customer inquiries (planned enhancement).
- **Secure Transactions**: Integration with payment gateways (future feature).

---

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: ASP.NET Core (C#)
- **Database**: (Planned) SQL Server or other database integrations
- **Version Control**: Git
- **Build Tool**: .NET Core SDK

---

## Getting Started
### Prerequisites
- .NET 6.0 SDK or higher
- A web browser
- Code editor like Visual Studio or VS Code

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ST10356476/Khumalo-Craft-Emporium.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Khumalo-Craft-Emporium/Khumalo_Craft_Emporium
   ```
3. Open the solution file in Visual Studio:
   ```bash
   Khumalo_Craft_Emporium.sln
   ```
4. Restore NuGet packages:
   - In Visual Studio, go to **Tools > NuGet Package Manager > Manage NuGet Packages for Solution** and restore dependencies.
5. Run the application:
   - Press `F5` or use the **Run** button in Visual Studio.
